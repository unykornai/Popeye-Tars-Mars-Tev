üöÄ Quickstart

Get your Unykorn L1 node running in under 5 minutes

1

Prerequisites

Make sure you have the following installed:

requirements
# Rust 1.75+ required $ rustc --version rustc 1.82.0 # Cargo package manager $ cargo --version cargo 1.82.0
üí° Installing Rust

If you don't have Rust, install it via rustup.rs: curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

2

Clone the Repository

terminal
$ git clone https://github.com/unykornai/Popeye-Tars-Mars-Tev.git Cloning into 'Popeye-Tars-Mars-Tev'... $ cd Popeye-Tars-Mars-Tev
3

Build the Project

Build all crates in the workspace:

terminal
$ cargo build --workspace --release Compiling tev v0.1.0 Compiling mars v0.1.0 Compiling tar v0.1.0 Compiling popeye v0.1.0 Compiling node v0.1.0 Finished `release` profile [optimized] target(s)
4

Run Tests

Verify everything works correctly:

terminal
$ cargo test --workspace running 10 tests (tev) running 15 tests (mars) running 12 tests (tar) running 8 tests (popeye) running 4 tests (node) test result: ok. 49 passed; 0 failed
‚úÖ All tests passing!

If you see 49 tests passed with 0 failures, you're ready to run a node.

5

Start a Development Network

Launch a local 3-node devnet:

powershell
PS> ./scripts/start-devnet.ps1 [INFO] Starting node-a on port 9001... [INFO] Starting node-b on port 9002... [INFO] Starting node-c on port 9003... [INFO] Devnet running! PIDs saved to logs/pids.json
‚ö†Ô∏è Windows Users

Run PowerShell as Administrator if you encounter permission issues with scripts.

6

Verify the Network

Run a smoke test to ensure nodes are communicating:

powershell
PS> ./scripts/smoke-test.ps1 [CHECK] Node A responding... ‚úì [CHECK] Node B responding... ‚úì [CHECK] Node C responding... ‚úì [CHECK] Peers connected... ‚úì [CHECK] Blocks propagating... ‚úì All smoke tests passed!
7

Stop the Network

When you're done, cleanly shut down the devnet:

powershell
PS> ./scripts/stop-devnet.ps1 [INFO] Stopping node-a (PID 1234)... [INFO] Stopping node-b (PID 1235)... [INFO] Stopping node-c (PID 1236)... [INFO] All nodes stopped.
